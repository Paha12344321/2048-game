# 🎮 2048 Game

A modern, responsive implementation of the classic 2048 puzzle game. Built for performance and smooth user experience.

## 🚀 Live Demo
(https://paha12344321.github.io/2048-game/)

## 🛠 Technologies
- React (v18)
- TypeScript
- SCSS (SASS)
- Vite (Build Tool)

## 🎯 Core Features
- Responsive design for mobile and desktop.
- Smooth tile animations and keyboard controls.
- LocalStorage integration to save your best score.
- Win/Loss state handling.

## 💻 Setup Instructions
1. Clone the repo: `git clone https://github.com/Paha12344321/2048-game.git`
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
4. Build for production: `npm run build`
